SketchでCSSプリプロセッサー向けのCSSスプライトミックスインコードを生成するプラグイン

これは何？

Sketch上のオブジェクトから、CSS PreprocessorのMixinsを生成するプラグインです。

動作環境

Sketch 3.2以降

ライセンス

MITライセンス

使い方

  1. アートボードを作成し、スプライトの元となるオブジェクトを作成します。
  2. コードを生成したアートボードを選択し、お使いのCSS Preprocessorに合わせてプラグインを実行します。
  3. 実行後はクリップボードにコピーされますので、ペーストして利用してください。

仕様

  • 選択しているアートボードからコードを生成します。プラグイン実行時にアートボードが選択されていない場合、レイヤーリストの最下層にあるアートボードからコードを生成します。
  • 画像までのパスと画像ファイルの拡張子は固定です。生成されたコードを必要に応じて修正してください。
